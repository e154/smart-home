## Log {#log}

Логирование состояния. Доступные методы приведены далее:

### .info() {#log_info}

```coffeescript
Log.Info('Info message')
```

### .warn() {#log_warn}

```coffeescript
Log.Warn('Warning message')
```

### .error() {#log_error}

```coffeescript
Log.Error('Error message')
```

### .debug() {#log_debug}

```coffeescript
Log.Debug('Debug message')
```
