## Node{} {#node}

Объект ноды
  
Доступные методы приведены далее:

### .Name() {#node_name}

Получить наименование устройства.

```coffeescript
name = node.Name()
```

**На выходе**

**Значение** | **Описание**
-------------|--------------
  `name`     | type: string
