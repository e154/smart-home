## Node{} {#node}

Объект ноды
  
Доступные методы приведены далее:

### .name() {#node_name}

Получить наименование устройства.

```coffeescript
name = node.name()
```

**На выходе**

**Значение** | **Описание**
-------------|--------------
  `name`     | type: string
